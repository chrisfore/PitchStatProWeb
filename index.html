Getting Started

Choose your sport (Baseball or Softball) to set up default pitch types and zones, then add your players and start tracking.

Track Tab

  • Select a player, pitch type, zone(s), and result
  • You can select up to 2 zones per pitch (e.g. Inside / Low)
  • Tap "Submit Pitch" to record — pitch type, zones, and result reset automatically
  • Live Results on the right show pitch breakdowns for the selected player
  • Color coding: green (65%+ win), yellow (48–64%), red (below 48%)
  • Use "Undo Last Pitch" to remove the most recent entry

Results Tab

  • View detailed statistics filtered by player or date range
  • Stats include strike%, ball%, foul%, hit%, and win% for each pitch/zone combo
  • Export PDF generates a printable report of the current view

Settings Tab

  • Add, rename, or remove players
  • Customize pitch types and zones to match your needs
  • Renaming a pitch type or zone updates all existing records

Data Management

  • Export Player — export a single player's data
  • Export All Data — export everything
  • Import Data — import a .pitchdata or .json file
  • Exported .pitchdata files are compatible with the Pitch Stat Pro iOS app

Data Storage

Your data is stored locally in your browser using localStorage. It persists across browser restarts and device reboots. However:

  • Clearing your browser data or site data will erase your data
  • Private/Incognito mode does not save data — it is wiped when the window closes
  • Different browsers maintain separate data (e.g. Chrome and Safari do not share data)
  • We recommend exporting your data regularly as a backup
